- Name
- URL

## 入口文档

- [平台总表（Markdown）](./sites.md)

## 渠道发现来源整合（LaunchDirectories）

- 来源： https://launchdirectories.com/
- 定位：聚合大量可提交的 startup directories / launch platforms
- 适用：需要批量找投放渠道的 Indie Hacker / SaaS / 工具类产品

### 可直接采用的实操结论

- 不要一口气全投，先选 10-15 个高相关渠道做高质量提交
- 目录站可分为免费提交和付费提交，先投免费且相关的
- 提交前统一准备素材，减少反复改稿和漏填

### 提交前素材清单

- 产品名、官网 URL、一句话描述
- 100-200 字简介
- Logo（建议 200x200 或更大）
- 截图或演示视频
- 定价信息
- 创始人/团队信息
- 社媒链接（如 X/LinkedIn）

### 工作流建议（适配本仓库）

1. 从聚合站筛出候选渠道
2. 在各渠道官网二次确认规则与是否仍可提交
3. 把确认后的渠道沉淀到 `docs/platforms/sites.md`
4. 提交前做去重，避免同一平台重复收录
