# Contributing

Thanks for contributing to Awesome Indie Launch.

## Contribution Principles

- Submit only verifiable resources
- Prefer official links
- Keep table format consistent and minimal
- For Reddit entries, keep topic labels clear and practical

## Fastest Way To Contribute

1. Edit the main lists directly:
   - `docs/platforms/sites.md`
   - `docs/reddit/subreddits.md`
2. Open a PR with clear source links in the description
3. Keep entries concise and verifiable

## Commit Message Suggestions

- `docs: add <platform-name>`
- `docs: add r/<subreddit>`
- `docs: update platform links`
- `docs: refine subreddit topics`
